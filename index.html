About DeepSight

DeepSight is a real-time deepfake detection system powered by a MobileNetV3 neural network with CBAM attention layers, trained to distinguish authentic faces from AI-generated or screen-reproduced media.

Every frame is analyzed using a combination of model confidence and image artifact signals — sharpness variance and RGB channel correlation — to reliably separate real faces from phone screen replays.

Version 1.0 · Built with TensorFlow, Flask & care.
